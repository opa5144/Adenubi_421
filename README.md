# Ordering System API
This RESTful API manages customers, orders and payments. It provides the ability to register customers and for customers to place orders and payments.

## Features
Lets you create, get, update and delete customers
Lets you create, get, update and delete orders
Lets you create, get, update and delete payments

## What It Uses
- Node.js
- MongoDB
- Swagger
